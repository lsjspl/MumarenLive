# 牧马人播放器             
只有直播的版本，其他多余的功能已经全部隐藏， 专供**老年人**使用

>增强了如下功能
* 自动缓存源
* 多源合并
* 频道分组
* 垃圾源屏蔽
* 频道信息Banner
* 删除多余功能

>源示例   
> 请参考我另外的项目
https://github.com/lsjspl/TV
